vpn_key Unlock 5,000 Requests & Saving

By default, anonymous users are limited to 60 requests per hour. By providing a GitHub token, your limit instantly upgrades to 5,000 requests per hour and unlocks the save button.

How to get a token:

  1. Go to GitHub ➔ Settings ➔ Developer Settings.
  2. Select Personal Access Tokens ➔ Tokens (classic).
  3. Click Generate new token (classic).
  4. Give it a name and check the repo box (required for saving).
  5. Click generate, copy the code, and paste it into the token field at the top.
